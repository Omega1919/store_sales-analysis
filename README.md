# 📊 Store Sales Dataset – Exploratory Analysis

## Project Overview
This project analyzes sales data from **827 retail stores** to understand the factors influencing store performance.

The analysis focuses on identifying relationships between:

- Store size
- Inventory levels
- Customer traffic
- Sales efficiency

The goal is to determine which factors contribute most to higher store revenue.

---

## Tools Used

- Excel
- Exploratory Data Analysis (EDA)
- Data Visualization
- GitHub

---

## Dataset Overview

The dataset contains performance metrics for **827 retail stores**.

| Column | Description |
|------|-------------|
| Store ID | Unique identifier for each store |
| Store Area | Size of the store in square yards |
| Items Available | Number of products available in the store |
| Daily Customer Count | Average number of customers per day |
| Store Sales | Total sales generated by the store |

### Data Quality

The dataset required minimal preprocessing:

- No missing values
- No duplicate rows
- Consistent numeric formatting

## Sales Distribution

Sales performance varies significantly across stores.

- A small number of stores generate **very high revenue**
- Some stores perform **far below the average**

**Insight:**  
Sales performance is uneven across the network, suggesting operational differences between stores.

---

## Customer Count vs Sales

Stores with the highest number of customers do **not always generate the highest sales**.

Some stores with fewer customers still achieve strong revenue.

**Insight:**  
Customer traffic alone does not explain sales performance.

---

## Store Size vs Sales

Stores were grouped by size:

| Store Size | Area Range | Total Sales |
|------------|------------|------------|
| Small | 775–1200 sq yards | ~7M |
| Medium | 1201–1700 sq yards | ~34.8M |
| Large | 1701–2229 sq yards | ~11M |

**Insight:**  
Medium-sized stores significantly outperform both small and large stores in total revenue.

---

## Inventory vs Sales

The number of items available in a store does not consistently correspond with higher sales.

Stores with both high and low inventory counts show varying sales performance.

**Insight:**  
Inventory size alone is **not a reliable predictor of revenue**.

---

## Customer Traffic by Store Size

Average daily customer traffic:

| Store Size | Avg Customers |
|------------|---------------|
| Small Stores | ~817 |
| Medium Stores | ~782 |
| Large Stores | ~779 |

**Insight:**  
Smaller stores attract slightly more customers, but this does not translate into higher sales.

---

## Sales Efficiency – Revenue per Customer

Sales efficiency was calculated using:v

```
sales per customer = store sales / daily customer count
```
Results:

| Store Size | Sales per Customer |
|------------|-------------------|
| Medium Stores | ~57,000 |
| Large Stores | ~16,000 |
| Small Stores | ~10,000 |

**Insight:**  
Medium-sized stores generate significantly more revenue per customer.

---

## Key Takeaways

- Sales performance varies widely across stores
- Customer traffic alone does not drive revenue
- Store size alone does not guarantee higher sales
- Medium-sized stores consistently outperform others
- Revenue per customer is a useful metric for measuring store efficiency

### Overall Insight

**Medium-sized stores provide the best balance between store size, customer traffic, and revenue efficiency, making them the strongest performers in this dataset.**
